# Roadmap

This document outlines planned features and improvements for FamilyFinance. Phases are not strictly sequential — contributions to any phase are welcome.

## Phase 1 — Core Improvements

- [ ] Additional file parsers (PDF statements, more bank-specific CSV formats)
- [ ] Batch transaction editing
- [ ] Transaction splitting (split one transaction across categories)
- [ ] API rate limiting and request throttling
- [ ] Pagination improvements and cursor-based pagination
- [ ] Duplicate transaction detection during import
- [ ] Transaction search with full-text search

## Phase 2 — Advanced Features

- [ ] Budget tracking and alerts
- [ ] Savings goals with progress visualization
- [ ] Recurring transaction detection and management
- [ ] Custom reports and date-range comparisons
- [ ] Multi-currency support with exchange rates
- [ ] Transaction tags (in addition to categories)
- [ ] Data export (CSV, PDF reports)

## Phase 3 — Integrations

- [ ] Plaid integration for automatic bank syncing
- [ ] Open Banking API support
- [ ] Webhook notifications for import events
- [ ] Email notifications and digests
- [ ] Calendar integration for bill reminders

## Phase 4 — Mobile and Platform

- [ ] Progressive Web App (PWA) support
- [ ] Offline mode with sync
- [ ] Push notifications
- [ ] Native mobile app (React Native)
- [ ] Multi-user household sharing with role-based permissions
